Exemplo de manipulação de DOM

Exercicio numero 1 - Iniciação de manipulação de DOM

Nesse exercicio você irá aprender a manipular o DOM, criando elementos, adicionando elementos, removendo elementos e alterando elementos.

Crie um botão que ao ser clicado mostra um alerta na tela dizendo: Botão foi clicado

Dica: Adicione um addEventListener ao botão / Pesquise sobre a função alert() para criar o alerta no navegador: https://www.w3schools.com/jsref/met_win_alert.asp

Exercicio numero 2 - Iniciação de manipulação de DOM

Crie um botão que ao ser clicado cria um novo elemento h1 com o texto "Danyela Ramos" e adicione esse elemento a div com o id de exercicío 2.

Dica: Adicione um addEventListener ao botão / Pesquise sobre a função createElement() e appendChild() para criar e adicionar o elemento ao body

Exercicio numero 3 - Iniciação de manipulação de DOM

Crie um botão que ao ser clicado altera o texto do elemento h3 dentro da div "exercicio3" para "Marilia Mendonça é Rainha".

Dica: Adicione um addEventListener ao botão / Pesquise sobre a propriedade innerText para alterar o texto dentro do elemento

Exercicio numero 4 - Iniciação de manipulação de DOM

Crie um botão que retira o ultima item da lista de nomes abaixo

  • João
  • Maria
  • Carlos
  • Julia
  • Lucas

Dica: Adicione um addEventListener ao botão / Pesquise sobre a propriedade lastChild para pegar o ultimo elemento da lista e removeChild() para remover o elemento

Bonus: Crie um botão que adiciona um novo nome a lista de nomes Bonus 2: Crie um botão que remove todos os nomes da lista Bonus 3: Crie um botão que altera o nome do primeiro item da lista para "Fulano" Bonus 4: Crie um botão que adiciona mais 5 nomes a lista
